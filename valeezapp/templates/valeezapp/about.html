About valeez

Thanks for stopping by!

My name is Eleanor, and I've been in the tech industry since 2008, as a consultant, client services director, and product executive. And I can code too. Give me coffee and an internet connection and I will figure anything out. More about me on my portfolio site.

In 2014, I took an evening course in Python and created the first version of valeez as a command line program (see image). In late 2015, I decided to take it further by rebuilding and refactoring it as and object-oriented program and a full-fledged web app using Django as the framework, which is what you see before you today. Learn more about how it works here.
